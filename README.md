# Gestion des réclamations

## Description

Ce projet est une application de gestion des réclamations développée en langage C. Il permet aux utilisateurs de soumettre, gérer, et suivre leurs réclamations tout en offrant aux agents et administrateurs des outils pour traiter ces réclamations de manière efficace.

## Prérequis

- Un compilateur C (par exemple, GCC)
- Un système Linux/Unix ou Windows avec un terminal compatible

## Installation

1. Clonez le dépôt:

   ```bash
   git clone https://github.com/Karim-Benkhira/Gestion_des_reclamations.git

   cd gestion-des-reclamations

   gcc Gestion_des_reclamations.c -o Gestion_des_reclamations

   ./Gestion_des_reclamations
